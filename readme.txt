Integrate a testimonial into your WordPress web site.

== Description ==

Awesome Testimonials : Allow Admin to customize testimonial, Easy to use and user friendly.Easy to add on your wordpress blog.

* Multiple transitions
* Responsive / Mobile Friendly
* Easy Customizable
* Easy to Use
* Autoplay option
* Testimonial with start ratings of user
* User designation/Post field is available
* Shortcode Powered
* Simple Testimonial Management Interface
* Most Browser Supported

== Installation ==

1. Upload `Awesome_Testimonial.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[pra_testimonial]` in your Pages.
