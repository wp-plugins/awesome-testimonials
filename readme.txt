Awesome Testimonials : Add an exited and user friendly Testimonial to Wordpress

== Description ==

Awesome Testimonials : Allow Admin to customize testimonial, Easy to use and user friendly.Easy to add on your wordpress blog.

== Installation ==

1. Upload `Awesome_Testimonial.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[pra_testimonial]` in your Pages.
